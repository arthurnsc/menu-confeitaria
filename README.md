# Menu de Confeitaria

Esta é uma aplicação web, que cria um cardápio digital de bolos e um painel administrativo para visualizar os pedidos.

## Tecnologias usadas

- Python
- Flask
- SQLite
- HTML
- CSS

## Funcionalidades

- Personalização, revisão e confirmação de pedidos.
- Painel para visualizar os pedidos.
- Armazenamento de pedidos em um banco de dados
- Banco de dados criado automaticamente.
- Layout responsivo.

## Como usar

- Clone o repositório
- Instale o Flask
- Rode o servidor

- Acesse no navegador:
Cliente - http://127.0.0.1:5000/
Administrador - http://127.0.0.1:5000/admin
