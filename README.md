# Menu de Confeitaria

Esta é uma aplicação web, que cria um cardápio digital de bolos e um painel administrativo para visualizar os pedidos.

## Tecnologias usadas

- Python
- Flask
- SQLite
- HTML
- CSS

## Funcionalidades
- Para clientes:
    - Personalização de pedidos
    - Revisão do pedido antes do envio
    - Confirmação do pedido.
    
- Para administradores:
    - Visualização dos pedidos
    - Confirmação de pedidos
    - Edição de pedidos
    - Exclusão de pedidos

- Acesso rápido ao painel e cardápio
- Armazenamento de pedidos em um banco de dados
- Banco de dados com criação automática
- Interface responsiva

## Como usar

- Clone o repositório
- Instale o Flask
- Rode o servidor
